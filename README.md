# Sistema de Locadora de Filmes

Projeto desenvolvido em Python para gerenciamento de uma locadora de filmes.

## Funcionalidades

- Cadastro de clientes
- Listagem de clientes
- Cadastro de filmes
- Listagem do catálogo
- Registro de aluguel
- Registro de devolução
- Pilha de devoluções (LIFO)
- Fila de reservas (FIFO)

## Estrutura do Projeto

```text
locadora_filmes/
│── main.py
│── clientes.py
│── filmes.py
│── dados.py
│── utils.py
```

## Tecnologias utilizadas

- Python 3

## Como executar

1. Abra o terminal na pasta do projeto
2. Execute:

```bash
python main.py
```
